Cara Penggunaan
  1. Pilih lembar kerja di sidebar (Claim 1, 2, 3, atau Progres 1)
  2. Klik tombol "+ Tambah Artikel" untuk menambah referensi baru
  3. Upload file PDF artikel jurnal — Claude AI akan mengekstrak metadata secara otomatis
  4. Lengkapi kolom analisis disertasi (Persamaan, Kelebihan, Kekurangan, Research Gap)
  5. Klik "Export Excel" untuk mengunduh file .xlsx yang identik dengan template asli
